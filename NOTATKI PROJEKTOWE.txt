---używane biblioteki---
laravel UI + moduł auth + nodeJS - rejestracja i logowanie

---czynności---
Dodanie bootstrapa z modułem logowania:
	composer require laravel/ui:^3.4
	php artisan ui bootstrap --auth
	instalacja node.js - https://nodejs.org/en/ - jest wymagana na komputerze
	npm install
	npm run dev

dodanie kontrolera użytkowników
	php artisan make:controller UserController --resource

szablon do listy użytkowników
	https://getbootstrap.com/docs/4.0/content/tables/

---lokalizacje---

routing: routes > view.php

css: public > css > app.css

kontrolery: app > http > contollers

menu dropdown: resources > views > layouts >app.blade.php

lista userów: resources > views > users > index.blade.php

role użytkowników funkcja ENUM:
app > enums > UserRole.php

sprawdzanie uprawnień:
backend: app > providers > AuthServiceProvider - Gate
frontend: resources > views > layouts >app.blade.php - @Can

blokada redirecta po rejestracji
